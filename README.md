# SIZZLD — Skill-Based Gaming Platform

Monorepo containing all three apps.

## Structure
- /admin   → Admin panel (React)
- /client  → User-facing app (React) — coming soon
- /server  → Backend API (Node.js + Express + Prisma)

## Quick start

# Backend
npm run dev:server

# Admin panel
npm run dev:admin

## Deployments (Vercel)
- Admin:  sizzld-admin.vercel.app
- Client: sizzld.vercel.app
- API:    sizzld-api.vercel.app
